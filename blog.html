  1. Variables declared with var are in the function scope. It can access from anywhere inside a function.
  2. Variables declared as let are in the block scope. In only can be accessed from a block where it was declared.
  3. Variables declared as const are in the block scope.
  • We cannot change value or variable type of a variable if it was declared as const but we can use let and var to declare a variable if we need to fully change the value or variable type.
  • Variable declared as var can be redeclared many times. This is why variable declared as let is recommended. For special purpose const is used.

Since regular functions are constructible, they can be called using the new keyword. However, the arrow functions are only callable and not constructible, arrow functions can never be used as constructor functions. Hence, they can never be invoked with the new keyword.

.map():
.map() executes the same code on every element in an array and returns a new array with the updated elements.

.forEach():
.forEach(), is used to execute the same code on every element in an array but does not change the array and it returns undefined.

.filter():
.filter() checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that fulfil the criteria.

.find():
.find(), is almost same as .filter(), just retun first matching element that fulfil the criteria.
